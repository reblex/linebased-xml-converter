# LineBased to XML Converter

Converts data of type:

P|förnamn|efternamn

T|mobilnummer|fastnätsnummer

A|gata|stad|postnummer

F|namn|födelseår

... to XML.

# Usage Example

./converter-osx-x64.exe example-input/data.txt -o data.xml

Use binary based on operating system.
